Hello Cryogen.. Goodbye Python.. Hello Clojure

it has been some time since I last wrote a blog post. I daresay, alot happened since then. I finished my exams YAY!.

anyway I am changing again the back end of this blog. not that I had much trouble wth jekyll. but long story short, I was learning Clojure and then I decided to make a blog and then I wanted an engine so I came back full circle to gh pages and that's it.

Clojure


Clojure is a functional language and a child of lisp that is based on the JVM.

clojure has a pragmatic approach to functional programming hence the java interop to make use of the big ammount of java libraries.


What I like about clojure

Clojure is fun

that's pretty much it. higher order functions are cool. functional programming makes for an elegant and concise code. I am not an expert on this matter and I don't do clojure for a living but I think I am being honest when I say I enjoy functional programming.

dynamic but not slow

clojure is dynamically typed which allows interesting stuff like the Replbut clojure still miles ahead of python in terms of speed while keeping its memory requirements slow.

more functions than python

this one is not based on actual data. but clojure is functional so it stresses on more functions and less types of data. and hence more common operations. for example to get the first item in an list in python you would write
alist[0]
and in a nested list
alist[0][0]
and here is the weird thing in clojure you would write
(first alist)
and in a nested list
(ffirst alist)
you are probaply thinking that this is redundant and you might be true but I think that if an operation is repeated so much time then it is worth it to make a shortcut for it.

innovative community

although I believe the clojure community is quite small, they seem quite enthusiastic about their language and they keep comming up with cool ideas and those cool ideas/libraries are what attracted me to clojue in the first place.

I would like to mention core.logic, core.match, core.typed, and clojurescript.
